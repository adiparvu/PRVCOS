What this is: Phase 12.3 — Document Sharing. Management page at /documents/shares (⌘K: "Document Sharing"). Two new tables — document_shares + document_share_access_log. Create an internal share (to a user) or an external secure link (unguessable token, optional expiry & password), with a permission level (view / download / edit / manage). Every external link view is recorded (count + last-accessed + IP). A pure helper derives each share's status (active / expired / revoked). The public resolver GET /api/share/[token] returns the document to a recipient and returns 410 once revoked or expired. No files are moved — links reference the existing document. Amber marks an expired link, per the design system.
