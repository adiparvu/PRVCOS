Architecture Overview
Implementation Plan
Layer File What it does
DB Schema packages/db/schema/user-preferences.ts New user_preferences table — typed columns, per-user unique constraint, cascade delete
Theme Types packages/ui/src/themes/index.ts TypeScript types: Theme, GlassStyle, AppearancePrefs, CSS variable maps
React Provider packages/ui/src/providers/appearance.tsx Context + useAppearance() hook. Applies data-theme / data-glass to <html>. SSR-safe. localStorage + server sync.
Settings UI packages/ui/src/components/appearance-settings.tsx Settings panel component. Theme picker + Glass style picker + Sync toggle. Full Liquid Glass.
API Route apps/web/src/app/api/preferences/route.ts GET + PATCH. Reads/writes user_preferences. Gate-chained. Audit-logged.
CSS Variables apps/web/src/app/globals.css Full [data-theme] + [data-glass] CSS variable system. Dark/Light/System × Translucid/Tinted/Adaptive.
Root Layout apps/web/src/app/layout.tsx Wraps with AppearanceProvider. Injects SSR-fetched prefs to avoid flash-of-wrong-theme.
Migration packages/db/migrations/ Auto-generated via pnpm db:generate after schema addition.
Database Schema
packages/db/src/schema/user-preferences.ts
export const themeEnum = pgEnum("theme", ["light", "dark", "system"])
export const glassStyleEnum = pgEnum("glass_style", ["translucid", "tinted", "adaptive"])

export const userPreferences = pgTable("user_preferences", {
  id:         uuid("id").primaryKey().defaultRandom(),
  userId:     uuid("user_id").notNull().unique().references(() => users.id, { onDelete: "cascade" }),
  theme:      themeEnum("theme").notNull().default("system"),
  glassStyle: glassStyleEnum("glass_style").notNull().default("adaptive"),
  syncEnabled: boolean("sync_enabled").notNull().default(true),
  updatedAt:  timestamp("updated_at", { withTimezone: true }).notNull().defaultNow(),
  syncedAt:   timestamp("synced_at", { withTimezone: true }),
})
