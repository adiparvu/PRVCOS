API Surface
GET
/api/projects
List all projects filtered by status (active | planning | review | done). Returns id, name, client, phase, budget, spent, workingDays, completionPct, teamIds, startDate, endDate.
GET
/api/projects/[id]
Full project detail: phases array, milestones checklist, team roster, budget breakdown, activity feed entries, linked documents, and client contact.
POST
/api/projects/[id]/phase
Advance or revert a project phase. Body: { action: "advance"|"revert", note? }. Writes audit log with actorId, companyId, resourceType: "project".
POST
/api/projects/[id]/flag
Flag a budget risk or issue. Body: { type: "budget_risk"|"delay"|"quality", severity, note }. Writes audit log. Triggers notification to company owner and manager.
