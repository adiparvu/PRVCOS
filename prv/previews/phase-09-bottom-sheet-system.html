Architecture

packages/ui — Base Primitives

  • BottomSheet — glass surface, grab handle, snap points
  • useBottomSheet — open/close/snap state hook
  • SheetStack — manages up to 3 stacked sheets
  • SheetPortal — renders via React portal into body
  • useSheetGesture — touch drag + spring physics
  • SheetContext — global open/dismiss from anywhere

Snap Point System

  • peek — 120px (quick preview)
  • mid — 50% viewport (detail + actions)
  • full — 92% viewport (scrollable list/form)
  • auto — height-content (action/confirm sheets)
  • Custom heights supported (px or %)
  • Velocity-aware snap — fast swipe overrides nearest

Gesture Physics

  • Spring: cubic-bezier(0.34,1.56,0.64,1)
  • Dismiss threshold: 40% down OR velocity > 1000px/s
  • Overscroll: rubber-band up to +20px
  • Velocity tracking: last 4 frames averaged
  • Backdrop blur: interpolates 0→20px with sheet position
  • Haptic feedback: open, snap, dismiss

App-level Sheet Variants

  • ActionSheet(items[], context?) — contextual actions
  • ConfirmSheet(title, body, onConfirm, variant?) — confirm/destructive
  • FilterSheet(fields[], onApply) — filter/sort builder
  • DetailSheet(entityType, entityId) — dynamic entity detail
  • useSheet() hook — open any sheet from any component
