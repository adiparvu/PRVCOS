What this is: Phase 21 — Tool Inventory. A read-only board at /analytics/tool-inventory (⌘K: "Tool Inventory"). Rolls the tool register into an availability view: the status mix (available / in use / maintenance / lost / retired), a utilization rate (in use ÷ operable stock), the lost-asset count, warranty exposure (expiring ≤30d / expired), and a per-category breakdown showing how many of each category are currently out. Amber/red flag the loss and warranty risks.
