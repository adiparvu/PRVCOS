1 Floating Tab Bar — 2 configs (Business OS · Public App) packages/ui New Floats 16px above safe area; content scrolls behind it; spring-animated active indicator
2 Bottom Sheet System — 8 variants packages/ui New All 8 types animate correctly; drag-to-dismiss at 60fps; spring physics on entry
3 Glass Context Menu — right-click + long-press packages/ui New Appears within 50ms; scale+opacity spring enter; dismiss on outside click or Esc
4 Command Palette — ⌘K, search, sections, keyboard nav packages/ui New Opens in <200ms; ↑↓ navigation; Enter confirm; Esc dismiss; real-time filter
5 Dynamic Island — compact + expanded + live activity hook packages/ui New Smooth size transitions via spring; role-specific data slots; updater hook exported

7 Navigation Shell — floating header + screen container packages/ui New Header floats over scroll content; glass tab bar + glass header rendered together
8 Storybook stories for all 7 new components packages/ui Stories All variants covered; dark+light theme switch works for each story
