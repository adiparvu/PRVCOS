H — Design Principles
Outline Default
All icons render as outlines at rest. Stroke weight 1.6px at 24pt. Filled variants reserved for selected/active states only.
Fill = Active
Filled variants signal the selected tab or active state. The fill color remains white — no color accents per B&W palette.
Destructive = Muted
Destructive actions use reduced opacity (35–50%), placed below separators. No red — stays within monochrome B&W palette.
Status: Subtle
Success, warning and info states use opacity variation, not color. Emergency alerts may use system colors sparingly.
Optical Scale
Icons scale optically: 16pt inline text, 22pt tab bars, 24pt default, 44pt widgets, 56pt module heroes.
One Language
Identical icons across Public App, Dashboard, Widgets, Dynamic Island, Live Activities, Sheets, Menus, Palette.
