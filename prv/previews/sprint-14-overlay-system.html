Component: GlassToast + useToast() hook — rendered via React portal into #toast-root.
Severities: success · error · warning · info — each has tinted icon + progress bar color.
Auto-dismiss: 3.5 s default (configurable). Progress bar depletes in real-time. Pause on hover.
Queue: up to 3 toasts stacked. Oldest exits when 4th appears. Spring entry + spring exit.
